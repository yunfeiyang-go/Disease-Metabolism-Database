This is a Curriculum Project in Database System Concepts.

Based on The Human Metabolome Database (HMDB) and Online Mendelian Inheritance in Man (OMIM) databases, important disease-related metabolic markers, information on metabolomic assays and supporting literature are collected and linked to human metabolic pathway maps (KEGG or REACTOME maps) with different colors to indicate rising or falling metabolites.

The source code of this project can be found on Github.
